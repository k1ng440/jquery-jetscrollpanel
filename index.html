    jquery.jetscrollpanel

    A simple and powerful jQuery plugin to hide native browser scrollbar

    Usage

    Inlcude jQuery and plugin files: 
    <link rel="stylesheet" href="jetscrollpanel.css">
<script src="jquery.min.js"></script>
<script src="jquery.jetscrollpanel.js"></script>
    And just write: 
    <script>
    $(function() { $("selector").jetScrollPanel(); });
</script>

    Examples

    Default

    $("#example1").jetScrollPanel();
